== Description ==
WP NTICStudio Backup est un plugin WordPress qui vous permet de sauvegarder facilement votre site web. Il offre une interface simple et intuitive pour créer et gérer les sauvegardes de votre site WordPress.

== Installation ==
1. Téléchargez le dossier `wp-nticstudio-backup` dans le répertoire `/wp-content/plugins/`.
2. Activez le plugin via le menu 'Extensions' dans WordPress.
3. Allez dans le menu 'Sauvegarde' dans l'administration WordPress pour configurer et gérer vos sauvegardes.

== Questions Fréquemment Posées ==
= Comment créer une sauvegarde ? =
Allez dans le menu 'Sauvegarde' dans l'administration WordPress et cliquez sur le bouton 'Créer une sauvegarde'.

= Où sont stockées les sauvegardes ? =
Les sauvegardes sont stockées par défaut dans le répertoire `wp-content/uploads/wp-nticstudio-backup`.
